## 客户端显示流量插件
- 安装在 x-ui所在的机器上
- 实时把流量使用情况写入备注栏，客户端通过订阅添加节点更新后可以查看当前流量使用情况
- 显示的数据是单个端口的总流量（不区分用户）
- 显示格式：

  节点备注_更新时间(D_H:M)_◔已用 ◕剩余（未设定上限时显示 99999）
